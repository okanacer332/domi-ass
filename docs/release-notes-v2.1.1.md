# Domizan v2.1.1

## Telegram Agent Duzeltmeleri

- `Mükellef sayısı kaç` gibi genel ofis sorulari artik yanlislikla bir mukellefe baglanmiyor
- `Meltem Gorenin VKN'si ne` gibi isimli sorgular daha dogru sekilde mukellefe eslesiyor
- kimlik sorularinda agent ozet yerine dogrudan ilgili VKN veya T.C. kimlik bilgisini veriyor
- genel agent cevaplarindaki metin dili sade ve tutarli hale getirildi
